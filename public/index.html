HIPAAA Custom Backend

This is an API-only project. There is no app at / beyond this page. Use the endpoints below (with netlify dev, base URL is http://localhost:8888).

Try it

  • GET /.netlify/functions/health — health check

Webhooks (POST)

  • /.netlify/functions/jotform-referral
  • /.netlify/functions/jotform-intake
  • /.netlify/functions/jotform-enrollment

Admin

  • GET /.netlify/functions/admin-review — header X-Api-Key
  • POST /.netlify/functions/admin-replay — header X-Api-Key
